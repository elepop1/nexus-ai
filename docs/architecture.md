## Components

### Frontend (Next.js)
- **Chat UI**: Real-time conversation interface
- **Image Upload**: Drag & drop image analysis
- **Document Processor**: PDF/text file Q&A
- **Reasoning Display**: Visual chain-of-thought

### Backend (FastAPI)
- **Chat Handler**: Session management, context building
- **Vision Handler**: Image processing & analysis
- **TTS Service**: Text-to-speech conversion
- **MiMo Client**: API communication layer

### MiMo V2.5 API
- **Chat Completions**: Text generation with reasoning
- **Vision**: Image understanding
- **TTS**: Natural voice synthesis

## Data Flow

1. User sends message via chat UI
2. Frontend calls backend API
3. Backend builds context from session history
4. Backend calls MiMo V2.5 API
5. MiMo processes request (reasoning + generation)
6. Response flows back through the chain
7. Frontend displays response with reasoning steps

## Security

- API keys in environment variables
- CORS restricted to frontend origin
- Rate limiting on all endpoints
- Input validation & sanitization
- No data persistence without consent
